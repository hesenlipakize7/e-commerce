I have developed an E-Commerce Backend Application using Java and Spring Boot.

The system supports user registration, product catalog management, shopping cart, order processing, and payment functionalities. For security, I implemented JWT-based authentication and role-based authorization. I also applied DTO architecture, MapStruct mapping, validation, pagination, and the Strategy Design Pattern to build a scalable and maintainable application.
